Project OverviewIn this simulation, we generate asset price paths where the variance evolves according to:$$\sigma_t^2 = \omega + \alpha \epsilon_{t-1}^2 + \beta \sigma_{t-1}^2$$We then perform a delta-hedging strategy on a European Call option. Crucially, the hedge is executed using the Black-Scholes delta, substituting the GARCH conditional volatility ($\sigma_t$) for the constant volatility parameter.Key Findings1. The Impact of $\alpha$ on Return DistributionsAs the ARCH parameter $\alpha$ increases, the "reactivity" of volatility to recent shocks intensifies. This results in:Leptokurtic Distributions: Higher $\alpha$ leads to "fatter tails" and "higher peaks" (excess kurtosis) in the return distribution.Extreme Events: There is a significantly higher probability of observing both near-zero price movements and extreme outlier events compared to a standard Gaussian distribution.
